# 📄 서울 다듬이(Seoul Dadeumi) 프로젝트 기술보고서

## 1. 개요 (Overview)
### 1.1 프로젝트 배경 및 목적
공공기관 및 행정 문서에서 사용되는 어려운 한자어, 일본식 표현, 권위적인 행정용어는 시민과의 소통을 저해하는 요소로 지적되어 왔습니다. '서울 다듬이'는 이러한 문제를 해결하기 위해 **AI 기술과 행정 데이터를 결합**하여, 누구나 클릭 한 번으로 공문서를 쉽고 바르게 고칠 수 있도록 돕는 스마트 문서 순화 솔루션입니다.

### 1.2 주요 목표
- **접근성 향상**: 어려운 행정 용어를 일상 언어로 즉시 전환
- **업무 효율화**: HWP/HWPX 문서의 자동 분석 및 수정 내역 개별 메모 삽입 자동화
- **지능형 순화**: 단순 단어 치환을 넘어 AI(Gemini)를 통한 문맥 최적화
- **데이터 기반**: 5,000건 이상의 방대한 행정순화어 DB 연동

## 2. 시스템 설계 및 기술 스택 (System Architecture)

### 2.1 기술 스택
- **Language**: Python 3.13
- **GUI Framework**: `CustomTkinter` (모던 윈도우 스타일 및 고대비 테마)
- **AI Model**: `Google Gemini 2.5 Flash API` (지능형 문장 다듬기)
- **Data Processing**: `Pandas`, `openpyxl` (대용량 엑셀 DB 처리)
- **Office Automation**: `pywin32` (한컴오피스 HWP/HWPX 제어)
- **Pattern Matching**: `Regex` (정규표현 기반 고속 탐지)

### 2.2 아키텍처 다이어그램 (핵심 프로세스)
1. **Input**: 사용자가 파일(TXT/HWP)을 선택하거나 텍스트를 붙여넣음
2. **Analysis**: 정규표현식 엔진이 사전 정의된 5,138건의 순화 대상어 검색
3. **Refinement**: 
   - **Lingo Match**: 데이터베이스 기반 단어 치환 제안
   - **AI Refinement**: Gemini 모델을 이용한 문장 구조 개선
4. **Output**: HWP 메모 기능을 활용한 수정 이력 삽입 및 최종 저장

## 3. 주요 기능 (Key Features)

### 3.1 듀얼 페인 하이브리드 에디터
- 원본 텍스트와 수정 제안을 나란히 배치하여 비교 분석 가능
- **양방향 동기화 스크롤(Synced Scrolling)**: 긴 문서를 검토할 때 양쪽 창의 위치를 자동으로 일치시켜 가독성 극대화

### 3.2 원클릭 대화형 치환 시스템
- 탐지된 단어를 클릭하면 즉시 순화어로 교체되며, 다시 클릭하면 원문으로 복원되는 인터렉티브 방식
- 대체 후보가 여러 개인 경우 팝업 메뉴를 통해 선택 가능

### 3.3 한글(HWP) 자동화 고도화
- **자동 메모 삽입**: 문서 내 단어가 수정된 위치에 한글 '변경메모'를 자동 생성하여 수정 근거를 명확히 제시
- **무한 루프 방지 로직**: 대량 수정 시 발생하는 한글 객체의 탐색 오류를 방지하기 위한 좌표 기반 방문 체크 알고리즘 적용

### 3.4 사용자 맞춤형 사전 관리
- 엑셀 일괄 업로드 기능을 통해 부서별, 업무별 특화 용어 사전 구축 지원
- `user_terms.json`을 통한 설정 자동 저장 및 관리

## 4. 기술적 차별성 (Technical Advantages)

| 기술 요소 | 상세 내용 |
| :--- | :--- |
| **고성능 대용량 정렬** | 5,138건의 단어를 탐지할 때, 긴 단어를 먼저 매칭하는 '최대 일치법(Longest Match)'을 적용하여 중복 치환 방지 |

| **디자인 접근성** | '서울색(모닝옐로우)'을 활용한 테마 설정으로 장시간 작업 시 눈의 피로도 저감 및 시인성 확보 |

## 5. 데이터베이스 관리 (Database)

- **Official DB**: 서울시 행정순화어 전체 목록을 포함한 5,138건의 대규모 데이터셋 탑재
- **Local Cache**: 빈번하게 수정되는 사용자 정의 단어의 경우 로컬 캐시를 통해 즉각 매칭 성능 향상
- **Template System**: 사용자 사전 확장을 위한 전용 엑셀 템플릿 생성 및 다운로드 기능 내장
